# MicroFlow Windows 安装说明

更新时间：2026-03-31

## 适用范围

本文适用于 Windows 10 / Windows 11 用户。

当前推荐发布物：

- `MicroFlow-Setup-x.y.z.exe`

如果后续保留便携版，也建议仅作为内测或排障用途，不作为普通用户主入口。

## 下载原则

- 仅从你的官方发布页、腾讯云 COS 下载页或你明确公布的下载地址获取安装包。
- 下载完成后，建议同步比对发布页提供的 `SHA256` 摘要。
- 若版本页中同时给出发布时间、版本号、摘要校验值，应三者一起核对。

## 安装步骤

1. 下载最新的 `MicroFlow-Setup-x.y.z.exe`。
2. 双击运行安装包。
3. 选择安装路径，建议保持默认。
4. 完成安装后，从开始菜单或桌面快捷方式启动。

## 如果出现 Windows SmartScreen 提示

由于当前项目暂未购买代码签名证书，首次运行时 Windows 可能提示“Windows 已保护你的电脑”。

这是未签名桌面软件的常见现象，不代表程序一定有问题。用户可按以下步骤继续：

1. 在提示窗口中点击“更多信息”。
2. 确认发布名称、版本号和下载来源无误。
3. 点击“仍要运行”。

如果你的发布页已提供 `SHA256`，建议先核对摘要后再继续。

## 覆盖升级

当前推荐升级方式：

1. 应用内检测到新版本。
2. 点击“下载更新”打开新版安装包下载页。
3. 下载最新安装包后直接覆盖安装。

通常无需手动卸载旧版本。

## 数据保留

- 覆盖安装不应删除用户已有的本地配置与数据库。
- 若后续版本涉及数据迁移，需在发布说明中明确标注。

## 排障建议

如果安装或启动失败，请优先检查：

- 是否从官方链接下载
- 是否被安全软件拦截
- 是否缺少系统权限
- 是否误用了过旧安装包

建议用户反馈以下信息：

- Windows 版本
- MicroFlow 版本号
- 安装包文件名
- 安装时间
- 错误截图

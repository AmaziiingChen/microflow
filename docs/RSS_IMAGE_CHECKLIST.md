# RSS 图片能力清单

更新时间：2026-03-30

## 文档目的

本文档用于记录 MicroFlow 中 RSS 图片能力的设计目标、开发顺序与当前进度。

这一轮图片优化的定位很明确：

- 不做“图片产品优先”
- 仍然以文字阅读为主
- 重点优化微信图文这种“文字 + 多张正文图”场景
- 目标是提升兼容性、阅读稳定性和界面完成度

## 设计原则

- 图片应服务正文，而不是压过正文。
- 正文图优先保留在文章流中，不重复堆成第二套图集。
- 对微信图片优先做稳定显示和失败兜底。
- 图片相关交互保持克制、整齐、轻盈，不做厚重相册风格。

# HTML 网页 AI 爬虫开发清单

更新时间：2026-03-30

## 当前判断

- 自定义数据源中的 HTML 网页 AI 爬虫主链路已经打通：
  - 前端规则配置
  - AI 规则生成与样本测试
  - 手动微调选择器
  - 规则保存 / 编辑 / 启停 / 删除
  - 调度器热加载
  - 动态抓取与文章入库
- 当前主要短板不在“有没有功能”，而在“泛化能力、稳定性、可观测性”。
- 现阶段不考虑旧测试数据兼容，优先把新链路做稳。
- Shadow DOM 当前能力边界：
  - 已支持 Playwright 浏览器抓取时展开 `open shadow root`
  - `closed shadow root` 仍无法通用抓取，这是浏览器机制限制

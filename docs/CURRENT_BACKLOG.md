# 当前待完善与优化清单

更新时间：2026-04-01

## 判断结论

MicroFlow 当前已经完成了三大主信息获取能力的主链路：

- 固定数据源订阅
- RSS 订阅
- 自定义 HTML 网页 AI 爬虫

现阶段的重点已经不再是“继续铺大功能面”，而是：

1. 收口稳定性
2. 收口数据结构与兼容逻辑
3. 补齐发布前必要能力
4. 为正式分发准备测试、更新和遥测基础

因此，后续工作建议按 `P0 / P1 / P2` 推进。

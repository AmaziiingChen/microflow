- 项目仍处于开发阶段，未正式发布。

## 当前重点

这一阶段的目标不是继续铺功能面，而是把已经打通的主链路做稳、做顺。

- [x] 补齐 RSS 图片加载失败时的前端降级和占位提示。
- [x] 增加站内图片放大 / 预览能力，减少跳浏览器查看原图的割裂感。
- [x] 明确 RSS 长文的折叠 / 展开策略，优化卡片和详情页阅读节奏。
- [x] 增强 RSS 规则预览态异常说明，补充正文缺失、正文偏短、图多文少和 AI 预览状态提示。
- [x] 统一 RSS 重新生成链路，使其尊重源级 AI 开关，并避免把增强正文误当成可编辑摘要。
- [x] 收敛 RSS 详情默认模式、手动保存总结和后台静默刷新之间的状态同步，减少阅读模式被意外重置的问题。
- [x] 已完成一轮 AI 摘要、详情载荷和编辑 / 重生成返回字段的统一收口。
- [x] 已完成一轮自定义规则生成、预览、保存链路的边界校验收口。
- [x] 已完成一轮启动性能和首屏响应优化，重点是规则懒加载与列表轻载化。
- [ ] 继续建立冷启动 / 热启动基线数据，量化启动性能。

# MicroFlow macOS 安装说明

更新时间：2026-03-31

## 适用范围

本文适用于 macOS 用户，优先支持 Apple Silicon 设备。

当前推荐发布物：

- `MicroFlow-x.y.z.dmg`
- 或 `MicroFlow.app.zip`

## 下载原则

- 仅从你的官方发布页、腾讯云 COS 下载页或你明确公布的下载地址获取安装包。
- 下载完成后，建议核对发布页提供的 `SHA256`。
- 若版本页中提供了发布时间、摘要校验值和版本号，建议一并检查。

## 安装步骤

### DMG 方式

1. 下载 `MicroFlow-x.y.z.dmg`。
2. 双击打开镜像。
3. 将 `MicroFlow.app` 拖入 `Applications`。
4. 打开“应用程序”启动 MicroFlow。

### ZIP 方式

1. 下载 `MicroFlow.app.zip`。
2. 解压后得到 `MicroFlow.app`。
3. 将应用拖入 `Applications`。
4. 从“应用程序”启动。

## 如果出现“无法打开，因为 Apple 无法检查其是否包含恶意软件”

由于当前项目暂未购买 Apple Developer 签名与公证服务，macOS 可能拦截首次启动。

用户可按以下方式放行：

### 方式一：右键打开

1. 在“应用程序”中找到 `MicroFlow.app`。
2. 右键点击，选择“打开”。
3. 在确认弹窗中再次点击“打开”。

### 方式二：系统设置中允许

1. 打开“系统设置”。
2. 进入“隐私与安全性”。
3. 在底部找到关于 `MicroFlow.app` 被阻止的提示。
4. 点击“仍要打开”。
5. 再次确认打开。

## 覆盖升级

当前推荐升级方式：

1. 应用内检测到新版本。
2. 点击“下载更新”打开新版下载页。
3. 下载新版 `dmg` 或 `zip`。
4. 用新版 `MicroFlow.app` 覆盖 `Applications` 中的旧版本。

## 数据保留

- 只要不主动删除应用支持目录，本地配置与数据库应保留。
- 若后续版本涉及迁移或兼容调整，需在发布说明中明确提示。

## 排障建议

若用户仍无法打开，请优先核查：

- 是否从官方地址下载
- 是否下载了错误架构版本
- 是否完整解压
- 是否已在“隐私与安全性”中允许

建议用户反馈以下信息：

- macOS 版本
- 芯片架构（Intel / Apple Silicon）
- MicroFlow 版本号
- 安装包文件名
- 报错截图
